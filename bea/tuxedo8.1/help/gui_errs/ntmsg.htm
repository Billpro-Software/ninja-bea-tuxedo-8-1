MESSAGE #7

ERROR: Cannot contact ipc server (r)

DESCRIPTION

An internal TUXEDO IPC error has occurred. The TransactNamedPipe() system function call which combines a write operation and a read operation into a single network operation from the the TUXEDO IPC Server Pipe (IPC_SERVER_PIPE) failed. The TUXEDO IPC server cannot be contacted.

ACTION

This function fails if the server did not create the pipe as a message-type pipe or if the pipe handle is not in message-read mode. Also, the function returns ERROR_MORE_DATA if the read buffer specified in the call is not large enough to hold the entire message. The extended error information is described in r by calling GetLastError(). Please check Microsoft documentation for details on those errors.
