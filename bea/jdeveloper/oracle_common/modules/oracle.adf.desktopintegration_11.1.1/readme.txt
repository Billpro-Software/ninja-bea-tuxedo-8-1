========================================
adfdi-excel-runtime-client-installer.zip
========================================

The archive located in this directory called 'adfdi-excel-runtime-client-installer.zip' 
contains all the files necessary to install the runtime edition of 
Oracle ADF 11g Desktop Integration for Excel.

This version of the client is intended to be used with the corresponding shared
library adf-desktop-integration.war located in this same directory.

For more information on this product, please visit the following link:

http://www.oracle.com/technology/products/adf/adfdi/index.html
